iWoT 入門教學
=======================================

2014年 W3C 聯盟正式成立 WoT Interesting Group。WoT 是物聯網的應用層，使用 Web 技術來打造物聯網應用，以 URL 來表示物聯網裝置，為物聯網設備加入 URL 的觀念。iWoT 平台以數據分析與智能預測為核心，支持多協議、多設備、多接入、視覺化開發特色，並符合國際 WoT 標準的物聯網開發與管理平台。
